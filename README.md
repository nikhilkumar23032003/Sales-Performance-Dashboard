# Sales-Performance-Dashboard

📝 Short Description

Interactive Power BI dashboard designed to analyze sales performance by tracking key KPIs such as Total Sales, Revenue Trends, and Quantity Sold. Incorporates data cleaning and transformation using Power Query, robust data modeling, and DAX measures to deliver accurate insights. Enables dynamic exploration across Region, Category, and Customer segments through intuitive and visually engaging reports, supporting effective data-driven decision-making.


🛠️ Tech Stack
Power BI
Power Query
DAX



📂 Data Source
Sample sales dataset (Excel/CSV)
Contains fields like Sales, Region, Category, Customer, and Order Details



🚀 Features
KPI tracking (Total Sales, Quantity Sold, Revenue Trends)
Data cleaning and transformation using Power Query
Data modeling with relationships across tables
DAX measures for performance metrics
Interactive slicers (Region, Category, Customer)
User-friendly and dynamic visualizations

📊 Example Insights
Identified top-performing regions and categories
Analyzed revenue trends over time
Compared customer segment performance

🎯 Outcome
Helps stakeholders monitor performance, identify trends, and make data-driven decisions efficiently.

❓Key Questions Addressed
Which region generates the highest sales revenue?
How is sales distributed across different product categories?
What is the overall trend of daily and monthly sales?
Who are the top customers contributing to revenue?
Which product category performs the best in terms of sales?
How does sales performance vary by region?
What is the total sales, quantity sold, and average sales value?
How can users interactively filter data by Region, Category, and Segment to uncover insights?
